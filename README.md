**Auteur :** Salaheddine Chouaref
**Établissement :** ISIB Bruxelles — Master Ingénieur Industriel
**Année :** 2025-2026

## Voir en ligne

https://chouarefsalaheddine.github.io/PortfolioSmals/

## Contenu

Le portfolio couvre les 11 sections de l'audit :

1. Statuts de l'entreprise
2. Comptes annuels (BNB)
3. Pitch de présentation
4. Mission, Vision & Valeurs
5. Business Model Canvas
6. Positionnement en matière d'innovation
7. Analyse de la concurrence
8. Segmentation, Ciblage, Positionnement, Différenciation
9. Cycle de vie d'un produit (eBox)
10. Diagnostic financier
11. Analyse SWOT

## Déploiement (instructions pour activer GitHub Pages)

1. Créer un compte GitHub avec le nom d'utilisateur **`chouarefsalaheddine`**.
2. Créer un dépôt public nommé **`PortfolioSmals`**.
3. Uploader `index.html` (et ce `README.md`) à la racine du dépôt.
4. Dans `Settings → Pages` → Source : `Deploy from a branch` → Branch : `main` → `/ (root)` → `Save`.
5. Le site sera publié sous quelques minutes à l'adresse `https://chouarefsalaheddine.github.io/PortfolioSmals/`.
