## Contenu

Le portfolio couvre les 11 sections de l'audit :

1. Statuts de l'entreprise
2. Comptes annuels (BNB)
3. Pitch de présentation
4. Mission, Vision & Valeurs
5. Business Model Canvas
6. Positionnement en matière d'innovation
7. Analyse de la concurrence
8. Segmentation, Ciblage, Positionnement, Différenciation
9. Cycle de vie d'un produit (eBox)
10. Diagnostic financier
11. Analyse SWOT
